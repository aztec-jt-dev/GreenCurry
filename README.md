# Run and deploy the Green Curry Hostel App

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
   
Gemini not needed, part of phase 2 build.

LIVE website / web app is hosted at https://greencurryhostel.click
